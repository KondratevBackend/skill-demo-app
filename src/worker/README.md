# Фоновые задачи (Taskiq)
В данном разделе проекта определяются планировщики (далее воркеры) — фоновые процессы для выполнения задач по расписанию.

[Документация Taskiq](https://taskiq-python.github.io)

### Область видимости (обнаружение воркеров)
Чтобы процессы планировщика и воркера смогли обнаружить ваши воркеры, соблюдайте соглашение об именовании файлов:
* Либо ```worker.py```
* Либо ```*_worker.py``` (например, ```clean_worker.py```)

Поиск файлов осуществляется рекурсивно по всем вложенным директориям. Например, файл по пути ```src/worker/shtab/test/worker.py``` будет успешно обнаружен процессами.
